Datatypes-----A datatypes define what kind of value of a variable can store and how the javascript engine treats that value in memory and operation.
Javascript is a dynamically typed language ,do not need to declare the data type expicitly. javascript engine automatically determine the type based on the assigned value.


Categories of JavaScript Data Types------------
JavaScript data types are divided into two main categories:

1.Primitive Data Types----Primitive data types are basic data types that store a single value.
They are immutable (their value cannot be changed directly).
JavaScript has 7 primitive data types.
(stack memory)
1. Number
2. String
3. Boolean
4. Undefined
5. Null
6. BigInt
7. Symbol

2.Non-primitive Data types---Non-primitive data types store collections of data and are stored by reference in memory.
Main types include:
(heap memory)
1. Object
2. Array
3. Function

Primitive vs Reference Data Types------------------------
This is one of the most important JavaScript concepts.

Primitive--------
Primitive values store the actual value directly in memory.
let a = 10;
let b = a;
b = 20;
console.log(a); // 10

Reference------
Reference types store the memory address instead of the actual value
let obj1 = { name: "Ronak" };
let obj2 = obj1;
obj2.name = "Riya";
console.log(obj1.name);-Riya

Shallow Copy

Definition
A shallow copy creates a new object or array but copies only the first level of properties.
If there are nested objects or arrays, their references are copied, not the actual values.

let obj1 = {
  name: "Ronak",
  address: { city: "Gwalior" }
};
let obj2 = { ...obj1 }; // shallow copy
obj2.address.city = "Delhi";
console.log(obj1.address.city);----------------Delhi


Deep Copy-----------
A deep copy creates a completely independent copy of the object, including all nested objects and arrays.
------
let obj1 = {
  name: "Ronak",
  address: { city: "Gwalior" }
};
let obj2 = JSON.parse(JSON.stringify(obj1)); // deep copy
obj2.address.city = "Delhi";
console.log(obj1.address.city);--------Gwalior
